# Ad Video Pipeline 🎬

Facebook 15s 竖屏广告视频全自动生成流水线，由 OpenClaw + Claude 驱动。

## 功能

- ✅ AI 自动生成广告脚本（5 Shot 结构，含口播+字幕）
- ✅ 脚本自动审核（17 条硬性标准，FAIL 自动打回重写）
- ✅ 即梦 AI 视频自动提交（Seedance 2.0 Fast / 9:16 / 15s）
- ✅ 视频自动下载 + 产品二次确认
- ✅ 片尾自动拼接
- ✅ 全程无需人工介入，完成后飞书通知

## 安装

1. 下载 [`ad-video-pipeline.skill`](ad-video-pipeline.skill)
2. 在 OpenClaw 中安装 Skill

## 使用口令




生成 easyfit 投放视频 5 个
生成 appediet 投放视频 10 个（含竞品分析）
新增产品
查看投放进度
